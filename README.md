# 🎓 Student Placement Analysis Dashboard

## 📌 Project Overview

The **Student Placement Analysis Dashboard** is an interactive Excel-based data analytics project designed to analyze placement trends, branch-wise performance, and skill gaps among students. The dashboard helps identify key factors influencing placements and provides insights for data-driven decision-making.

## 🎯 Problem Statement

Educational institutions often struggle to track student placement performance and identify the skills that impact employability. This dashboard aims to provide a centralized and interactive solution to monitor placement statistics, compare branch-wise outcomes, and highlight areas where students need improvement.

## 🛠️ Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Timeline Filter
* KPI Cards
* Excel Formulas & Functions
* Dashboard Navigation

## 📊 Dashboard Highlights

* Overall Placement Summary
* Branch-wise Placement Analysis
* Skill Gap Analysis
* Gender-wise Placement Distribution
* Average Package & Highest Package Tracking
* Interactive Filtering using Slicers and Timeline

## 📂 Dataset

* **1000 student records**
* 20+ data fields including Branch, Gender, CGPA, Skills, Certifications, Internships, Projects, Placement Status, Company, and Package.

## 💡 Key Insights

* Branch-wise comparison helps identify departments with higher placement rates.
* Students with stronger technical and communication skills tend to have better placement outcomes.
* Certifications, internships, and project experience positively influence placement success.
* Interactive filters enable dynamic analysis across multiple dimensions.
